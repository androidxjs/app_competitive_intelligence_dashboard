# Glossary — App Competitive Intelligence

| Term | 中文 | Definition |
| --- | --- | --- |
| Owned App | 自有 App | 用户管理的己方 App，例如 B612 咔叽，也可以是其他 App。 |
| Competitor | 竞品 | 当前自有 App 的竞品 App。一个竞品可以被多个自有 App 复用。 |
| Channel | 渠道 | App Store 中国区、华为、小米、OPPO、vivo、应用宝等商店或分发渠道。 |
| Snapshot | 快照 | 某个 App 在某个渠道某次抓取到的版本、描述、截图、评分、评论等数据。 |
| Insight | 洞察 | 从评论、版本、截图、价格、官网变化中提炼出的结构化发现。 |
| Requirement Candidate | 候选需求 | 由洞察转出的产品需求草稿，包含问题、证据、影响和建议优先级。 |
| Evidence | 证据 | 评论、页面快照、截图、价格变化、更新日志等可回溯来源。 |
| Feature Matrix | 功能矩阵 | 当前自有 App 与竞品的功能差距、优势和跟进建议。 |
| Report | 周报 / 报告 | 按周期汇总竞品变化、评论热点、候选需求和证据的输出。 |
